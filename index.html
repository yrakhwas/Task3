Linear Equations

A linear equation is any equation that can be written in the form ax + b = 0 where a and b are real numbers and x is a variable. This form is sometimes called the standard form of a linear equation. Quadratic Equation

The standard form of a quadratic equation looks like this: ax2 + bx + c = 0 where a, b, c are known values and x is the variable or unknown (we don't know it yet). Also, a can't be 0.

Intersection

The intersection of two sets A and B, denoted by A ∩ B, is the set of all objects that are members of both the sets A and B. In symbols, A ∩ B = { x : x ∈ A and x ∈ B } That is, x is an element of the intersection A ∩ B if and only if x is both an element of A and an element of B.
